このサイトについて

このサイトは MyToybox の App Clip / Universal Link 用ランディングです。デモごとの URL は /<screen-id>/ 形式です（例: /ringSliderScreen/）。

Associated Domains 用の apple-app-site-association は Koshimizu-Takehito/.well-known リポジトリで配信しています。

新しいデモページを追加するときは、my-toybox の RouteCatalog.availableRoutes の Screen.rawValue と同じディレクトリ名で index.html（layout: clip_landing）を置いてください。
